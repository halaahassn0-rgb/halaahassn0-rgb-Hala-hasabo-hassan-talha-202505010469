# Food Delivery Receipt System

## Purpose

This application generates a receipt for customers ordering food.

## Tech Stack

- Python 3
- Functions
- Modules

## How to Use

1. Run main.py.
2. Enter customer details.
3. The receipt will be displayed automatically.
